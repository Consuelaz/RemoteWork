MCP Protocol Design Flaw Exposes 200K+ AI Servers to Remote Code Execution

AI圈再掀安全风暴。4月15日，网络安全公司OX Security发布重磅报告：Anthropic主导的MCP（模型上下文协议）存在严重设计缺陷，可导致远程代码执行。这一漏洞影响范围极广，波及超过20万台配置了MCP的AI服务器，包括Cursor IDE、Claude Code等主流开发者工具。这意味着攻击者可能在开发者不知情的情况下，通过恶意MCP服务器远程操控AI Agent执行任意代码。对于正在大规模部署AI Agent的企业而言，这无疑是一记警钟。Anthropic尚未对此作出正式回应。

OX Security researchers discovered a critical design flaw in Anthropic's Model Context Protocol that allows remote code execution on over 200,000 AI servers. The vulnerability affects major development tools including Cursor IDE and Claude Code, potentially giving attackers control over AI agents. Anthropic has not yet released a formal response to the vulnerability disclosure.

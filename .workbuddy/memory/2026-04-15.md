# 2026-04-15 日记

## 移动端导航栏修复

### 问题
手机浏览器看不到导航链接（岗位列表、100+公司列表、AI资讯、加入社群）。
根因：`style.css` 的 `@media (max-width:768px)` 中有 `.nav-links { display: none; }` 直接隐藏了导航，且没有汉堡菜单替代方案。

### 修复内容
1. **style.css**：删除 `display: none`，改为汉堡菜单折叠交互
   - 新增 `.nav-toggle` 汉堡按钮样式（三横线 → X 动画）
   - 手机端 `.nav-links` 折叠为绝对定位下拉菜单（max-height 动画展开）
   - 手机端 `.nav-cta`（"发布岗位" 按钮）隐藏
2. **index.html**：加入 `#navToggle` 汉堡按钮 + `toggleNav()` JS
3. **companies.html**：同上，补充了缺失的 "AI资讯" 导航链接
4. **community.html**：同上，补充了缺失的 "AI资讯" 导航链接
5. **ai-news/index.html**：加 `@media` 块，手机端缩减 padding，优化 controls 布局

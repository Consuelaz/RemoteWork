万亿 AI 巨头排着队 IPO——但钱可能先到你口袋

本文由 AI 辅助整理，观点源于独立开发实战与真实 builders 的一手声音。
"AI is now a profit generator."（AI 现在是印钞机了。）

这是黄仁勋今天上午在台北 Computex 主题演讲上的原话。他站台上，背后是一台需要两千人才能推得动的 Vera Rubin AI 平台。

同一周，OpenAI 秘密提交了 IPO 招股书，估值破万亿美元。Anthropic 在洽谈 300 亿美元新融资，SpaceX 给它签了 450 亿美元的算力合同。DeepSeek 把旗舰模型 API 永久砍到四分之一的价格。Cursor Composer 2.5 性能追平 Opus 4.7，价格只要十分之一。

看起来是一个属于超级巨头的时代。

但另一组数据：2026 年新注册公司中，36.3% 是单人创办的。AI Agent 把单人创业的年成本从 22.5 万美元打到了 1.8 万，降幅 92%。

巨头在造 AI 工厂，独立开发者在用 AI 工厂的边角料造自己的生意。问题是你站哪边。

01 / 降价不是价格战，是算力民主化

5 月 22 日，DeepSeek 宣布 V4-Pro API 永久降价 75%：输入 0.435 美元/百万 token，输出 0.87 美元/百万 token。比 GPT-5.5 便宜约 98%。

这已经不是"促销"了，这是结构性改变。更值得关注的是降价的底气：V4-Pro 全部跑在华为昇腾芯片上，芯片成本结构和英伟达路线完全不同。

与此同时，Cursor 发布了 Composer 2.5——基于月之暗面开源的 Kimi K2.5（万亿参数 MoE 架构），SWE-Bench Multilingual 79.8%，定价只有 Opus 4.7 的十分之一。阿里 Qwen3.7-Max 在 GPQA 和数学推理上双双超越 Opus 4.6 Max。智谱 GLM-5.1 输出速度达到 400 tokens/s，刷新全球纪录。

信号很清楚：顶级模型能力不再被一两家公司垄断。 中国厂商走了一条不同的路——华为芯片 + 开源权重 + 极致性价比。

对独立开发者这意味着什么？以前你做 AI 产品，模型成本是最大的不确定因素。现在你可以用 DeepSeek 做批量处理，用 Cursor Composer 做代码生成，用 GLM-5.1 做需要速度的场景。模型层正在变成水电煤——你不会在意用的是哪家发电厂的电。

02 / AI 编程工具从"帮你写"变成了"替你干"

上周 Google I/O 发布了 Jules——一个在后台异步运行的编码 Agent。你下指令，它自己开 VM、克隆仓库、读代码、写方案、改文件、跑测试、提 PR。支持 Python、TypeScript、Go、Rust、Java。

这不是 Copilot 那种"你写一半它补全"的辅助模式。这是"你说需求它去执行"的代理模式。

Claude Code 快速模式最近默认切到了 Opus 4.7，代码迭代速度提升 2.5 倍。Anthropic 还花了 3 亿美元收购 Stainless（SDK 生成工具），收购后直接关停所有托管产品，切断竞对 SDK 供应——这不是买技术，是买供应链控制权。

Cursor 母公司 Anysphere 的估值在 16 个月内从 25 亿飙到 600 亿（SpaceX 招股书披露的收购期权价）。一个 AI 编程工具值 600 亿美元——市场在告诉你，编程这件事本身正在被重新定义。

对独立开发者：你不再是"写代码的人"，你变成了"定义需求的人"。 过去你一个人写不完一个全栈产品，现在 Jules 和 Claude Code 可以同时给你跑 5 个 PR。但这也意味着，"会用 AI 写代码"这个技能溢价正在快速归零。真正的护城河是你的判断力——你比别人更早知道用户要什么、不需要什么。

03 / 巨头在烧钱抢人才，但人才红利在往你这边漏

这周最戏剧性的事：Karpathy 加入了 Anthropic。

作为 OpenAI 联合创始人、"Vibe Coding"概念的提出者、AI 圈最有影响力的技术布道者，他在 5 月 19 日深夜发了四句话官宣。一小时内 300 万浏览。

他去的不是 OpenAI——他去了 Anthropic。

这不是孤例。Workday CTO、Instagram CTO、Box CTO 此前都已以个人贡献者身份加入了 Anthropic。而 OpenAI 同时在流失高管：产品负责人 Kevin Weil 和技术负责人 Bill Peebles 离开，COO 被调去"特殊项目"，Sora 被关停。

大公司烧钱抢人，对独立开发者意味着什么？意味着两个东西在往你这边漏。

第一，技术外溢。Karpathy 加入 Anthropic 预训练团队，他的研究最终会变成 Claude 的能力提升——然后你通过 API 用上。就像 Karpathy 本人说的，他"依然对教育事业充满热情"。他教的东西，你学得到。

第二，市场教育。Anthropic 的 Managed Agents、Google 的 Jules、OpenAI 的 Codex——巨头在花钱教育企业客户"AI Agent 能干什么"。等企业接受了这个概念，最灵活、最能定制化服务他们的，是独立开发者，不是巨头。

巨头做基础设施，你做最后一公里。

04 / 这几天 builders 圈值得关注的事

NVIDIA Vera Rubin 平台发布。 黄仁勋今天在 Computex 发布了进入量产的 Vera Rubin AI 平台，定位为"代理式系统"（agentic system）。他明确说 NVIDIA 已经从 GPU 公司进化为"全栈 AI 基础设施公司"。对独立开发者：这意味着算力基础设施会进一步加速普及。你不需要买，你需要会调用。

Anthropic 收购 Stainless 后关停全部服务。 3 亿美元收购 SDK 生成工具，收购后直接切断竞对供应。这不是"生态开放"的打法，这是供应链封锁。信号：Anthropic 正在从"模型公司"变成"平台公司"。独立开发者如果重度依赖 Anthropic 生态，需要留意这个变化——平台公司最终都会收"平台税"。

Google I/O 发布了 ADK 1.0。 同时支持 Python、TypeScript、Java、Go 的 Agent 开发框架 1.0 稳定版。超过 200 万 token 上下文窗口。对独立开发者：如果你还没试过 Agent 框架，ADK 的多语言支持让 Java/Go 开发者零门槛上手，是把现有后端能力 Agent 化的最快路径。

05 / 如果你今天只做一件事

第一件事：选一个 AI 编程 Agent（Jules/Claude Code/Codex），下周开始让它独立完成一个你拖了很久的小功能。 不是让它帮你写——是让它自己去读代码、改代码、跑测试、提 PR。你做 code review。这是一种全新的工作方式，你越早习惯当"管理者"而非"执行者"，你的产出天花板就越高。

第二件事：算一笔你的模型成本账。 DeepSeek V4-Pro 降价后，检查你的 API 开销——是不是有些场景不需要用最贵的模型？按任务类型做模型分层：批量处理用便宜模型，复杂推理用强模型，代码生成用专用模型。省下的不是钱，是你试错的胆子。

第三件事：关注 Anthropic 平台化趋势。 它收购 Stainless 后关停竞对供应，这件事说明它正在从"卖模型"往"锁生态"转型。如果你的产品核心依赖 Claude，考虑增加一个备选模型通道。独立开发者的优势是灵活，不要主动放弃这个优势。

意外的利好：NVIDIA 在造 AI 工厂，但工厂越高效，原材料（算力）越便宜，加工工具（模型）越便宜——最后受益的，不是开工厂的，是在工厂旁边开小店的。

当一件商品的制造成本趋近于零，价值就转移到"知道该造什么"的人手里。你不需要建工厂，你只需要知道你的用户想要什么。

这恰好是独立开发者最擅长的事。
